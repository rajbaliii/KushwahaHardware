# Kushwaha Hardware Premium 🛠️

A state-of-the-art enterprise resource planning (ERP) and billing solution tailored for **Kushwaha Hardware**, Mahanwa, Bihar. Designed to transform local business operations with modern technology.

![App Vision](https://via.placeholder.com/800x400?text=Kushwaha+Hardware+Premium+ERP)

## 🌟 Vision
To provide a seamless, intuitive, and feature-rich digital experience that empowers Kushwaha Hardware to manage inventory, sales, and customer relations with world-class efficiency.

## 🚀 Key Features

### 📊 Intelligent Dashboard
- **Visual Analytics**: Real-time sales trends and profit charts using MPAndroidChart.
- **Smart Alerts**: ⚠️ Instant notification for low-stock items.
- **Financial Pulse**: Summary of today's collections and pending receivables.

### 📦 Dynamic Inventory
- **Emoji-Categorized**: Easily navigate through 🎨 Paints, 🔧 Tools, 🏗️ Iron, and more.
- **Barcode System**: 🏷️ Generate unique barcode serials in bulk and export to printable PDF sheets.
- **Integrated Scanning**: Rapid inventory updates and sales using the internal barcode scanner.

### 🧾 Account Book (Khata)
- **Unified Management**: Single interface for both **Customers** and **Suppliers**.
- **Historical Ledger**: Chronological tracking of all sales, purchases, and payments ("You Got" / "You Gave").
- **Smart Reminders**: Direct WhatsApp integration for pending dues.

### 💳 Professional Billing
- **Premium Invoices**: Brand-consistent PDFs with integrated **BharatPe QR Codes** (BHARATPE.8F0V1P1U0V14772@fbpe).
- **Multi-channel Sharing**: Direct WhatsApp sharing for invoices.
- **Flexible Payments**: Robust management for Cash, Credit, and Digital payments.


### 🔐 Security & Reliability
- **Biometric Guard**: 🛡️ Industry-standard fingerprint/face unlock for data privacy.
- **Cloud Shield**: (Planned) Automated encrypted backups to Google Drive.
- **Local-First**: Works flawlessly offline, syncing data instantly when needed.

## 🛠️ Technical Excellence
- **Frontend**: Jetpack Compose (Material 3) with custom micro-animations.
- **Backend Architecture**: MVVM + Clean Architecture for maximum scalability.
- **Data Layer**: Room Database with multi-table indexing for high performance.
- **DI**: Hilt for robust dependency management.
